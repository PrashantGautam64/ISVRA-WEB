ISVRA Tech Website

Official multi-page website for ISVRA Tech, a web development studio founded by engineering students.

This project is an example of contemporary frontend design, responsive design, and well-structured, maintainable code written using HTML, CSS, and Vanilla JavaScript—no frameworks or external UI libraries.

⸻

Overview

The website is an embodiment of the studio’s services, portfolio, development process, company philosophy, and contact information.

It is a fully static, lightweight, and optimized website for performance and accessibility.

⸻

Tech Stack
	•	HTML5 — semantic structuring
	•	CSS3 — custom styling with variables, responsive design, and smooth transitions
	•	Vanilla JavaScript (ES6+) — interactivity, scroll animations, and form validation
	•	Google Fonts — typography (Inter)

No frameworks or external dependencies except Google Fonts.

⸻

Structure

index.html      Home
work.html       Portfolio
services.html   Services
process.html    Development process
about.html      Company information
contact.html    Contact and FAQ

style.css       Global styling
script.js       Interactive behavior

Key Features
	•	Multi-page responsive design, mobile-first design
	•	Sticky navigation with active page highlighting
	•	Smooth scroll and scroll-triggered animations
	•	Client-side form validation with error/success handling
	•	Accessible semantic structuring with keyboard navigation and proper color contrast

⸻

Deployment

The website can be hosted on any static hosting service (Netlify, Vercel, GitHub Pages,etc..)
